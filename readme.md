## 📖 Sobre el Proyecto

**KuyénDev** es el sitio web oficial de los servicios de soporte informático profesional fundado por **Franco Luna** en Osorno, Región de Los Lagos.

La plataforma está diseñada para ofrecer una experiencia de usuario fluida, moderna y accesible, conectando a personas, pymes y establecimientos educacionales con soluciones tecnológicas reales. Nuestra filosofía principal es clara: **optimizar y recuperar antes de reemplazar**.

## ✨ Características de la Web

- **Diseño Moderno y Responsivo:** Interfaz adaptada a cualquier dispositivo móvil, tablet o escritorio.
- **Animaciones Dinámicas:** Elementos interactivos, micro-interacciones y fondos animados mediante Canvas (`bgCanvas`).
- **Navegación Intuitiva:** Menú estructurado (Nosotros, Servicios, Educación, Metodología, FAQ).
- **Integración con WhatsApp:** Llamados a la acción directos para agendar diagnósticos o solicitar soporte remoto/presencial.

## 🛠️ Servicios Principales

KuyénDev abarca todo el espectro del soporte tecnológico:

- 💻 **Formateo Profesional e Instalación de SO**
- ⚡ **Optimización de Rendimiento de Equipos**
- 🛡️ **Eliminación de Virus y Malware**
- 📦 **Instalación y Activación Permanente de Microsoft Office**
- 🔒 **Seguridad Digital y Ciberseguridad**
- 🎓 **Talleres Educativos y Optimización de Laboratorios (Colegios/Liceos)**

## ⚙️ Tecnologías Utilizadas

Este proyecto se desarrolló utilizando tecnologías web nativas para garantizar un rendimiento óptimo y tiempos de carga instantáneos:

- **HTML5:** Estructuración semántica.
- **CSS3:** Estilos personalizados, gradientes y animaciones (sin librerías externas pesadas).
- **JavaScript (Vanilla):** Lógica de interacción, menú responsivo y animaciones de fondo (Canvas).

## 🚀 Metodología de Trabajo

1. **Diagnóstico:** Análisis completo del estado actual.
2. **Análisis:** Identificación de la causa raíz.
3. **Propuesta:** Opciones claras y precios transparentes.
4. **Implementación:** Ejecución cuidadosa y precisa.
5. **Optimización:** Ajustes finales para un rendimiento máximo.
6. **Seguimiento:** Aseguramiento de calidad post-servicio.
